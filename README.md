<h1 align="center"> CouchPotato </h1>
<br/>

<p> A extension of the FoodieSpot project, created for users to upload their recipes as well as search and favorite recipes.<p/>

<h2> Built With </h2>
<br/>

- React
- JSON
- CSS
- Mongoose
- Mongodb
- node.js
- express.js
- Axios
  - API Weather(Open Meteo)
